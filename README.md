# SE181

SE181 project

# Installation steps

1. Install Node https://nodejs.org/en/
2. `npm install -g ionic`
3. cd into folder contain `src/`
4. `npm install`
5. `ionic serve `

# Creating pages

1. cd into folder contain `src/app`
2. `ionic generate page <pageName>`
3. cd into `src/<pageName>`
