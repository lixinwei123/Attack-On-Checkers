# Installation steps

1. Install Node https://nodejs.org/en/
2. cd into folder contain `SE181-Checker/src/`
3. `npm ci`
4. `ionic serve `

# Creating pages

1. cd into `SE181-Checker/src/`
2. `ionic generate page <pageName>`
3. cd into `src/<pageName>`
