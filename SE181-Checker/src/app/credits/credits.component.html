Why did you make this?

This project was made in parallel with SE 181 at Drexel University, a software engineering course that introduces advanced software development fundamentals including memory management, typing and scoping, datastores, software testing, and security, as well as user-centric design and user experience. This course will be taught using a specified programming language of instruction.
